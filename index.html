Experiencias de RA habilitadas para la web

AR.js Studio es una plataforma de creación de RA de código abierto para crear experiencias personalizadas de realidad aumentada. Puede crear aplicaciones de RA basadas en la ubicación o en los marcadores e implementarlas directamente en la web. Eso significa que no tendrás que descargar ninguna aplicación adicional para acceder a tu experiencia. Trabajar con AR.js Studio tampoco requiere ningún conocimiento de codificación. Puede comenzar a crear su primer proyecto AR ahora mismo.
